# PocketSense AI - Hackathon Presentation

This document contains the presentation slide content outlining the advanced, AI-driven capabilities of PocketSense AI. It is designed to be concise and impactful for a hackathon pitch.

---
## Slide 1: Advanced Features

* 🚨 **Smart Alerts System**
  * Detects unusual spending patterns
  * Triggers budget overspending warnings automatically
* 💡 **AI Insights Engine**
  * Performs spending personality detection
  * Analyzes and visualizes weekly spending trends
* 🎯 **Smart Savings Planner**
  * Generates daily savings predictions
  * Provides goal tracking along with progress metrics
* 🧑 **Profile System**
  * Features a fully personalized dashboard 
  * Allows seamless monthly budget setting

---
## Slide 2: System Architecture

* **Frontend:** HTML, CSS, JavaScript (Vanilla UI)
* **Backend:** Java Spring Boot APIs
* **Database:** Supabase PostgreSQL
* **Authentication:** Supabase Auth
* **System Flow:**
  * User → Frontend → Backend → Supabase → AI Logic → Dashboard

---
## Slide 3: Application Workflow

* **1.** User Login via Supabase Auth
* **2.** User inputs new expense
* **3.** Backend processes transaction data
* **4.** AI Engine generates:
  * Real-time alerts
  * Personalized insights
  * Tailored savings plan
* **5.** Dashboard state updates in real-time

---
## Slide 4: AI-Based Intelligence

* **Rule-Based Intelligent System:**
  * Core engine driving dynamic financial advice
* **Anomaly Detection:**
  * Flags transactions where expense > 2x user average
* **Spending Personality Analysis:**
  * Categorizes users as *Foodie Spender*, *Weekend Spender*, or *Saver*
* **Predictive Savings Calculation:**
  * Forecasts future savings based on historical spend velocity

---
## Slide 5: Future Enhancements

* 🎤 **Voice-Based Expense Entry:** Hands-free logging via Speech-To-Text
* 🤖 **AI Chatbot Financial Assistant:** Interactive Q&A for finance
* 🏦 **Bank API Integration:** Plaid/Open Banking auto-sync
* 🎮 **Gamification:** Badges and rewards for strict budgeting
* 📊 **Advanced Analytics Dashboard:** Deeper graphical reporting

---
## Slide 6: Conclusion

* PocketSense AI serves as a smart, automated financial coach
* Proactively helps students control daily spending
* Provides actionable, AI-driven behavioral insights
* Encourages the development of better long-term saving habits

> *"Not just tracking expenses — guiding financial decisions."*
