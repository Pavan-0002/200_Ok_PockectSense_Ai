## 🚀 Features

* 💸 Expense tracking
* 💰 Savings tracking with progress
* 🚨 Real-time on-screen alerts (toast)
* 🧑 Profile with photo upload
* 🎯 Goal & budget tracking

---

## 🧠 Smart Features

* ⚠️ Unusual spending detection
* 📊 Dynamic insights
* 💡 Daily savings calculation

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Spring Boot
* Database: Supabase (PostgreSQL + Storage)
* Hosting: Netlify (frontend), Railway/Render (backend)

---

## ▶️ How to Run

### Backend

```bash
.\mvnw spring-boot:run
```

### Frontend

Open `index.html` in browser
